PRODUCT OVERVIEW
Arc Pass is a credential + claim platform where users can check eligibility and claim one of two passes:
1. Founder Pass
2. Builder Pass

Founder Pass is invite-only.
Builder Pass is claimable through verified builder proof, with tiers assigned based on contribution.

The app must support:
- public eligibility checking
- account login
- pass preview
- pass download
- onchain minting as NFT
- manual wallet-address delivery option
- light and dark theme
- responsive design
- strong UX
